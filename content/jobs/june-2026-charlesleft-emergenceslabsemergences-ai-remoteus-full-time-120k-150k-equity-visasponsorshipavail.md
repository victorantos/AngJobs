Emergences Labs (emergences.ai) - AI &#x2F; Systems Engineer — NeoHuman - REMOTE (US) - Full-time - $120K-$150K + equity - Visa sponsorship available

We&#x27;re a small, research-driven, VC-backed team building NeoHuman: a desktop-first system that observes how people *actually* use AI tools, files, and workflows during real tasks, and turns those sessions into reliable evaluation evidence. We care about the process, not just the final answer — what someone typed, what they copied, what they verified, how they moved across apps.

We&#x27;ve published the AgentIF-OneDay benchmark. Now we&#x27;re scaling the capture and evaluation infrastructure underneath it.

You&#x27;d own:
- Cross-platform desktop capture across macOS + Windows (AX&#x2F;UIA trees, OCR, clipboard, keyboard&#x2F;mouse boundaries, file interactions)
- Trace fusion — turning noisy low-level signals into clean human-readable session timelines
- The evaluation evidence pipeline (data model, ingestion, exports)
- LLM normalization that summarizes traces *without* hallucinating or hiding evidence quality
- Privacy&#x2F;consent&#x2F;proctoring safety for enterprise customers

We want an AI-native engineer:
- 3+ years shipping production TypeScript + Node.js
- Built or shipped Electron + native helpers
- Strong PostgreSQL (schema design, indexing, data quality)
- Comfortable with OS APIs, permissions, background processes
- Hands-on LLM API experience (structured output, evaluation, hallucination control)
- Uses Claude Code, Cursor, or similar as a daily working medium — not a side experiment
- Bonus: macOS AX &#x2F; ScreenCaptureKit &#x2F; CGEventTap, Windows UIA, or you&#x27;ve studied&#x2F;built things like Screenpipe, OpenAdapt, Rewind, OpenChronicle

Stack: TypeScript · Node.js · Electron · native macOS&#x2F;Windows · Next.js 15 · React 19 · Postgres + Supabase · OCR · Accessibility APIs · OpenAI &#x2F; Anthropic &#x2F; Gemini · Vercel · GitHub Actions

Why us: Hard, unusual problem at the boundary of OS internals, multimodal capture, and AI evaluation. Small team, technical founders who code daily, real ownership of the capture stack. We hire on demonstrated capability, not credentials.
