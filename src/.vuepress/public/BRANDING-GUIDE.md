# AngJobs Branding Assets

Complete branding package for AngJobs.com - Your source for tech jobs from Hacker News "Who is hiring?" threads.

## Brand Identity

### Colors
- **Primary Orange**: #FF6600 (Hacker News signature orange)
- **Secondary Orange**: #FF8833 (gradient accent)
- **Dark Background**: #2C3E50
- **Text Gray**: #7F8C8D
- **Light Gray**: #95A5A6

### Typography
- **Primary Font**: Segoe UI, Arial, sans-serif
- **Logo Weight**: 700 (Bold)
- **Subtitle Weight**: 400 (Regular)

### Design Concept
The AngJobs logo features a stylized "AJ" monogram with:
- An upward arrow symbolizing career growth and opportunity
- Clean, modern lines representing the tech community
- Hacker News orange as a nod to the source platform
- Professional aesthetic suitable for job seekers and employers

## Usage Guidelines

### Website Integration

#### Favicon
```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/favicon-simple.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/icon-512.png">
```

#### Logo in Header
```html
<img src="/logo.svg" alt="AngJobs - Tech Jobs from Hacker News" height="60">
```

### Social Media

#### Twitter/X
- **Profile Picture**: Use `profile-400.png`
- **Banner**: Use `twitter-banner.png` (1500×500px)
- **Name**: AngJobs
- **Bio**: "Tech jobs from Hacker News 'Who is hiring?' threads • Monthly updates • angjobs.com"

#### LinkedIn
- **Logo**: Use `profile-400.png`
- **Cover Photo**: Use `twitter-banner.png` (may need adjustment)

#### Facebook
- **Profile Picture**: Use `profile-400.png` or `profile-800.png`
- **Cover Photo**: Create custom 820×312px from twitter-banner template

### Print Materials
- Use `logo.png` (1024×1024px) or `logo.svg` for scalability
- Maintain minimum clear space around logo (equal to height of the "A")
- Preferred backgrounds: white, dark (#2C3E50), or light gray (#ECF0F1)

### Don'ts
- Don't alter the color scheme
- Don't distort or stretch the logo
- Don't use on busy backgrounds that reduce legibility
- Don't separate the icon from wordmark in primary logo
- Don't add effects (shadows, glows, bevels)

## File Formats

### SVG (Scalable Vector Graphics)
- **Best for**: Web, print, any scalable use
- **Advantages**: Infinite scalability, small file size, editable
- **Use when**: Quality and scalability are priorities

### PNG (Portable Network Graphics)
- **Best for**: Social media, raster graphics
- **Advantages**: Universal support, transparency
- **Use when**: Platform doesn't support SVG

### ICO (Icon Format)
- **Best for**: Browser favicons
- **Advantages**: Multi-size support, legacy browser compatibility
- **Use when**: Serving favicons for websites
