Formal - Multiple Positions - REMOTE - Full-Time - $100k - $250k + Equity

We&#x27;re building a new computing stack with formally verified data, memory, and resource isolation between executions for instant, truly elastic global compute. This includes designing and building a production language in a way that&#x27;s never been done before, with complete, mechanized formal specifications and a certifying compiler from the start. Our immediate goals include an alternative to eBPF and building the world&#x27;s first serverless networking infrastructure.

We are VC funded with runway for multiple years. The team is composed of PhDs from Stanford, UW, OSU, and Brown, advised by professors from MIT and UWaterloo.

We are currently hiring for the following positions:

- [1] Staff Software Engineer: Compilers, Programming Languages, and Verification ($175k - $250k + ≥ 0.4%)

- [2] Formal Verification Engineer: Formal Methods and Programming Languages ($120k - $200k + ≥ 0.2%)

- [3] Software Engineer: Compilers and Programming Languages ($100k - $175k + ≥ 0.1%)

- [4] Formal Methods PhD Intern: Formal Methods and Programming Languages ($10k &#x2F; month)

Please see [5] for general information. Apply via our job board [6].
