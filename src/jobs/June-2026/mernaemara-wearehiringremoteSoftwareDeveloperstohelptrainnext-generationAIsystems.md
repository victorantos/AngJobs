Python, Rust, Go, Java, Node.js, full-stack development, Android, or iOS.

- how you structure clean, maintainable code
- how you evaluate edge cases and failure scenarios

- examples from real projects where you fixed bugs, improved performance, or made architecture decisions

The interview is not only checking whether you can code. It is also checking how you think, explain, prioritize, and handle ambiguity.
