MintMCP - Founding Engineer &#x2F; Founding GTM - SF Bay Area (ONSITE)

MintMCP helps enterprises safely do more with Agents (e.g, governance around MCP tools and agents like Claude Code &#x2F; Cowork, and others).  We&#x27;re building an MCP gateway as the foundation with the proper abstractions that enterprises need to allow long running agents to be safely deployed across an entire organization; organizations can easily and clearly limit the blast radius of what an agent can do, while giving it the right combination of tools to be useful.

We&#x27;re a small team of humans with dozens of agents: our company operates on the infrastructure we build for enterprises.  We&#x27;re looking for agent-forward engineers and GTM folks to help solve scaling problems, maintain the highest standards for security, make agents and MCPs usable and useful.

Problems we&#x27;re solving right now: Agent Identity, ReBaC for tool resources, scalable hosting of private MCP servers, and our agent platform that vertically integrates with the foundational MCP gateway layer.

If you have experience in infra, security, and agents, and MCPs, you might find the problems we&#x27;re working on exciting!
